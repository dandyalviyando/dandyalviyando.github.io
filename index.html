My Android Projects

Trans Metro Bandung application was made with the purpose of research about "Integration of Payment Monitoring & Bus Position Monitoring via Android App Based on Internet of Things". This project uses a combination of several components, including GPS U-BLOX Module NEO-6M, Logitech C270 Webcam and android device. Trans Metro Bandung application is made using android studio with Java programming language. For authentication and data storage, this app uses several features of Firebase.

This app has been tested using ISO/IEC 25010 standard. The application quality test results show that the application is feasible to use. Click Here to check information about this project research.
